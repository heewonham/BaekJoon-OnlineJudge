### 기존에 생각했던 방법

: x의 범위를 가운데 패티를 기준으로 나누어 생각한다.

- 가운데 패티 이전 범위 -> 이전 단계의 햄버거에서 x-1장 까지 패티 개수 계산
- 가운데 패티 == x -> 이전 단계 햄버거 패티 개수 + 1
- 가운데 패티 이후 -> 이전 단계 햄버거 패티 개수 + 1 + 이전 단계의 햄버거에서 x-1장 까지 패티 개수

그런데 오답이었음..
